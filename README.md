# Low-Code 低代码可视化编辑平台

一个基于 Vue 3 的可视化低代码页面编辑器，拖拽组件即可搭建页面，支持导出可运行的 Vue 源码。

## 功能特性

- **37 个组件**：覆盖基础、表单、布局、高级、图表、数据展示 6 大类
- **拖拽搭建**：从左侧拖入组件，画布内拖拽排序，布局组件支持嵌套
- **属性配置**：右侧面板实时编辑组件属性、样式，所见即所得
- **事件联动**：支持点击、值变化、加载时触发，可控制组件显隐、设置变量、修改属性、重置表单
- **导出源码**：一键导出 Vue 3 SFC 单文件组件（含 Element Plus + ECharts），开箱即用
- **实时预览**：生成独立 HTML 页面在新窗口预览效果
- **撤销重做**：基于命令模式，支持 100 步操作回退
- **表达式引擎**：支持 `{{ }}` 模板语法绑定动态数据
- **画布辅助**：设备切换（PC/平板/手机）、网格参考线、缩放、框选

## 技术栈

| 技术 | 说明 |
|------|------|
| Vue 3 | 前端框架（Composition API + `<script setup>`） |
| Vite 8 | 构建工具 |
| Pinia 3 | 状态管理 |
| Element Plus 2 | UI 组件库 |
| ECharts 6 | 图表渲染 |
| vue-draggable-plus | 拖拽排序 |

纯 JavaScript 项目，无后端，纯客户端实现。

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

在编辑器中拖拽搭建页面，点击工具栏「导出源码」即可下载 `.vue` 文件。

## 核心设计

- **JSON 驱动**：所有组件以 JSON 树形结构存储，增删改查、撤销重做、导入导出都围绕 JSON 操作
- **渲染与配置分离**：画布渲染器使用原生 HTML/CSS，避免表单交互干扰编辑；导出和预览使用真实 Element Plus 组件
- **递归组件树**：`RenderComponent` 自引用实现嵌套渲染，布局组件可无限嵌套
- **命令模式**：撤销/重做基于 Command 模式，支持操作合并（快速连续编辑合并为一步）

## 组件一览

| 分类 | 组件 |
|------|------|
| 基础（7） | 按钮、文本、输入框、图片、分割线、图标、链接 |
| 表单（10） | 下拉选择、单选框、多选框、开关、文本域、日期选择、时间选择、评分、滑块、上传 |
| 布局（6） | 容器、行、列、卡片、标签页、折叠面板 |
| 高级（9） | 表格、轮播图、进度条、徽标、标签、提示、头像、面包屑、分页 |

| 数据（3） | 统计数值、倒计时、树形控件 |

## 事件联动

每个组件可配置多个事件，每个事件包含触发器和动作列表：

**触发器**：点击（click）、值变化（valueChange）、加载时（mount）

**动作**：
- 显示/隐藏组件
- 设置变量
- 修改组件属性
- 重置表单

## 项目结构

```
src/
├── main.js                              # 应用入口
├── App.vue                              # 根组件
├── assets/theme.css                     # 设计令牌（CSS 变量）
├── config/components.js                 # 组件注册表（37 种组件配置）
├── stores/
│   ├── editor.js                        # 编辑器状态（组件树、选中、历史）
│   ├── dataStore.js                     # 运行时数据（表单、变量、联动）
│   └── history/                         # 命令模式撤销/重做
├── utils/
│   ├── codeGenerator.js                 # 导出 Vue SFC 源码
│   ├── previewGenerator.js              # 生成预览 HTML
│   ├── expressionEngine.js              # {{ }} 表达式引擎
│   └── helper.js                        # 工具函数
├── views/Editor.vue                     # 编辑器主页面
└── components/
    ├── layout/                          # 编辑器 UI（工具栏、面板、配置表单）
    └── render/                          # 画布渲染器（6 个分类渲染器）
```
