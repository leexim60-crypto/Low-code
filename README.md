  一个基于 Vue 3 的可视化低代码页面编辑器，拖拽组件即可搭建页面，支持导出可运行的 Vue 源码。

  功能

  - 37 个组件：基础、表单、布局、高级、图表、数据展示 6 大类
  - 拖拽搭建：从左侧拖入组件，右侧实时配置属性、样式、事件
  - 事件联动：支持点击、值变化、加载时触发，可控制组件显隐、设置变量、修改属性
  - 导出源码：一键导出 Vue 3 SFC 单文件组件，开箱即用
  - 实时预览：生成独立 HTML 页面预览效果
  - 撤销重做：基于命令模式，支持 100 步操作回退
  - 表达式引擎：支持 {{ }} 模板语法绑定动态数据

  技术栈

  Vue 3 + Vite + Pinia + Element Plus + ECharts

  使用

  npm install
  npm run dev

  在编辑器中拖拽搭建页面，点击工具栏「导出源码」即可下载 .vue 文件。
